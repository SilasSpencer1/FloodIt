# FloodIt
This is a model version of the game flood-it, a single-player game
where the player is presented with a size [n x n] board containing an n amount of colors.
The goal of the player is to turn the board into one singular color by clicking on connecting colors to gradually
changing the color of the overall board.

This game imports the HashMap class and the imp WOLRD class which is unique to Northeastern Universities computer sciecne curriculum
